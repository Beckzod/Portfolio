About me

Software Engineer

I have been programming in C/C++ for six years, including bachelor degree tenure. During the academic years, the degree curriculum covered many courses which involved C/C++ programming such as Introduction to Programming in C, Data Structures, Object Orient Programming, System Programming, Computer Graphics, etc. Some of these courses included C/C++ projects as an essential part of the course. In Data Structures, our team of two students had developed Airline Management System in C. In the project, we developed a software which possessed features such as airline registration, management of arrival and departure of planes, ticket booking facility, weather details, etc. The project employed various concepts of Data Structures such as linked lists, queues, searching and sorting algorithms, etc. Graphics.h library was used for coding user interface, data was stored in and retrieved from binary file by using fstream.h and iostream.h library. We used structural programming methodology and subroutines were coded for various operations. The project size was more than thousand lines of code.
